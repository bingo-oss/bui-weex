## bui-weex

> BUI-Weex 是一套专门为 Weex 前端开发者打造的一套高质量UI框架。为了达到更好的效果，请在 WeexSDK 0.11.0 + 上使用这套UI框架，目前仅支持iOS和android平台，后续将针对web平台进行降级支持。

官网：[http://dev.bingocc.com/buiweex/](http://dev.bingocc.com/buiweex/)

想要看到最新的问题fix情况，请切换到 `dev`分支
